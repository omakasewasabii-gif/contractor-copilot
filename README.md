# NutriServe — Institutional Food Service & EPISD Procurement Demo Platform

## 🟢 Status: SELECTION-READY (RFP 26-027 COMPLIANT)

This platform is a high-fidelity demonstration suite designed for the **EPISD Food & Nutrition Services** procurement committee (RFP 26-027).

### Core Hardened Features:
- **Bilingual PoS**: Real-time EN/ES terminal with negative balance hardening.
- **Forensic Audit**: SHA-256 adjustment logs and TDA reimbursement certification.
- **Cluster Monitoring**: 69-site rollout simulation and network resilience testing.
- **Parent Transparency**: Allergen tracking and household meal application portals.
